* Library for read and write jpeg encoded file

DICOM library use IJG library with patch that support lossless JPEG,
which wrote by Ken Murchison. The libjpeg with patch can be downloaded
from sf.net. ( http://sourceforge.net/projects/jpeg/files ) 

1. Download ljpegsrc.v6b.zip from http://sourceforge.net/projects/jpeg/files/ftp.oceana.com/ 
2. run do_copy.sh script
